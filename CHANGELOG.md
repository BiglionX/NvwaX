## 版本说明

### 版本号规则

格式：`MAJOR.MINOR.PATCH`

- **MAJOR**：不兼容的 API 变更
- **MINOR**：向后兼容的功能新增
- **PATCH**：向后兼容的问题修正

### 变更类型

- **Added**：新增功能
- **Changed**：现有功能变更
- **Deprecated**：即将删除的功能
- **Removed**：已删除的功能
- **Fixed**：错误修复
- **Security**：安全性修复
