## 🔍 技术细节

### 1. 高亮算法

**步骤**:
1. 转义特殊字符（避免正则注入）
2. 创建全局不敏感正则
3. 用正则分割文本
4. 遍历片段，匹配则包裹 `<mark>`

**安全性**:
```typescript
// 转义特殊字符
const escapedQuery = query.replace(/[.*+?^${}()|[\\]\\\\]/g, '\\$&');
```

**性能**:
- 时间复杂度: O(n)，n 为文本长度
- 空间复杂度: O(n)，存储分割后的数组
- 适合短文本（< 1000字符）

---

### 2. 搜索历史存储

**数据结构**:
```typescript
// localStorage
{
  "bounty_search_history": [
    "客服",
    "订单查询",
    "数据分析"
  ]
}
```

**操作**:
- **添加**: 头部插入，去重，截取前10个
- **删除**: 过滤掉指定项
- **清空**: 移除 localStorage key

**优化**:
- Lazy initialization（懒加载）
- 避免 useEffect 中 setState
- 减少不必要的渲染

---

### 3. React 最佳实践

**Lazy State Initialization**:
```typescript
// ✅ 推荐：在初始化时读取
const [history, setHistory] = useState<string[]>(() => {
  const saved = localStorage.getItem(STORAGE_KEY);
  return saved ? JSON.parse(saved) : [];
});

// ❌ 不推荐：在 useEffect 中设置
useEffect(() => {
  const saved = localStorage.getItem(STORAGE_KEY);
  setHistory(JSON.parse(saved)); // 触发额外渲染
}, []);
```

**优势**:
- 减少一次渲染
- 避免闪烁
- 更好的性能
