# Agent 广场页面更新说明

## 📝 更新内容

已修复 Agent 广场页面，现在同时支持显示**单个智能体（Agents）**和**团队技能（Team Skills）**。

---

## 🎯 新增功能

### 1. 分类筛选器增加"智能体"选项

现在分类筛选器包含：
- **全部** - 显示所有智能体和团队技能
- **智能体** ⭐ - 仅显示单个 AI Agent
- **虚拟公司** - 仅显示虚拟公司模板
- **开发团队** - 仅显示开发类团队技能
- **数据分析** - 仅显示分析类团队技能
- **内容创作** - 仅显示内容类团队技能

### 2. 智能体列表展示

当选择"智能体"或"全部"分类时，会显示：

**蓝色渐变横幅**：
```
🤖 智能体
发现和探索优秀的单个 AI Agent
```

**智能体卡片信息**：
- 名称
- 来源标签（GitHub/HuggingFace）
- 描述
- 星级评分 ⭐
- 下载次数 ⬇️
- 标签（Tags）
- 查看详情链接（跳转到外部仓库）

### 3. 团队技能列表展示

当选择"虚拟公司"或其他团队分类时，会显示：

**紫色渐变横幅**（仅虚拟公司）：
```
🏢 虚拟公司
组建你的 AI 团队，像真实公司一样协作工作
```

**团队技能卡片信息**：
- 名称
- 分类标签（虚拟公司有特殊紫色标识）
- 描述
- 角色数量 👥
- 分类标签
- 查看详情链接（跳转到详情页）

## 🔍 数据源

### 智能体（Agents）
- **API**: `GET /api/search/agents?q=ai agent&page=1&limit=30`
- **数据来源**: GitHub、HuggingFace 爬虫
- **显示条件**: 选择"全部"或"智能体"分类

### 团队技能（Team Skills）
- **API**: 
  - `GET /api/team-skills/marketplace?page=1&limit=30`（全部）
  - `GET /api/team-skills/category/:category?page=1&limit=30`（按分类）
- **数据来源**: 数据库 `team_skills` 表
- **显示条件**: 选择"全部"或任意团队分类

## 🚀 使用方法

### 查看智能体
1. 访问 http://localhost:3000/marketplace
2. 点击"智能体"分类按钮
3. 浏览单个 AI Agent 列表
4. 点击"查看详情"跳转到 GitHub/HuggingFace

### 查看虚拟公司
1. 访问 http://localhost:3000/marketplace
2. 点击"虚拟公司"分类按钮
3. 浏览三个虚拟公司实例
4. 点击"查看详情"（详情页待实现）

### 查看全部
1. 访问 http://localhost:3000/marketplace
2. 默认显示"全部"分类
3. 先显示智能体列表，再显示团队技能列表
