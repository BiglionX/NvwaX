## 🎯 快速开始

1. 打开浏览器，访问：`http://localhost:3000/bounties`
2. 浏览所有开放的悬赏任务
3. 使用搜索和过滤功能找到感兴趣的悬赏

## 🔍 搜索功能详解

### 1. 全文搜索

**使用方法**:
- 在搜索框输入关键词（如"客服"、"订单"）
- 500ms 后自动搜索
- 搜索结果会高亮显示匹配的关键词

**示例**:
```
输入: "客服"
结果: 
- AI[客服]智能体系统  ← 黄色高亮
- [客服]订单查询助手   ← 黄色高亮
```

---

### 2. 搜索建议（Autocomplete）⭐

**使用方法**:
- 输入 1-2 个字符
- 自动显示相关建议（最多5条）
- 使用键盘 ↑↓ 选择，Enter 确认
- 或直接用鼠标点击建议

**键盘快捷键**:
- `↑` / `↓` - 上下选择建议
- `Enter` - 确认选择
- `Esc` - 关闭下拉框

**示例**:
```
输入: "客"
显示建议:
- AI客服智能体系统
- 客服订单查询助手
- 智能客服机器人
```

---

### 3. 搜索历史

**功能**:
- 自动保存最近 10 次搜索
- 点击历史记录快速重新搜索
- 悬停显示删除按钮
- 点击"清空"删除所有历史

**示例**:
```
📜 搜索历史                    [清空]

[客服] [订单查询] [数据分析]
   ↑ 点击重新搜索  × 删除单条
```

---

### 4. 热门搜索

**功能**:
- 基于最近 30 天悬赏统计
- 显示 Top 8 热门关键词
- 5分钟自动更新
- 点击立即搜索

**示例**:
```
📈 热门搜索

[#1 客服] [#2 订单] [#3 数据] [#4 API]
[#5 代码] [#6 数据库] [#7 智能] [#8 助手]
```

---

### 5. 技能过滤

**可用技能**:
- 客服 (customer-service)
- 数据库 (database-connector)
- 代码生成 (code-generation)
- 数据分析 (data-analysis)
- API集成 (api-integration)

**使用方法**:
- 点击技能下拉框
- 选择需要的技能
- 立即过滤结果

---

### 6. 状态过滤

**可用状态**:
- 开放中 (open) - 默认
- 已领取 (claimed)
- 待验证 (submitted)
- 已完成 (completed)
- 已取消 (cancelled)

**使用方法**:
- 点击状态下拉框
- 选择要查看的状态
- 立即过滤结果

---

### 7. 组合搜索

**强大功能**:
可以同时使用多个过滤条件

**示例**:
```
搜索: "订单" + 技能: "database-connector" + 状态: "open"

结果: 开放中的、需要数据库技能的、与订单相关的悬赏
```

## 💼 发布悬赏

### 前提条件

- 必须登录账户
- 账户有足够的积分余额

### 发布步骤

1. 点击页面右上角的 **"+ 发布悬赏"** 按钮
2. 填写表单：
   - **标题**: 简明扼要描述任务
   - **描述**: 详细说明需求和要求
   - **技能**: 添加所需技能标签（可多选）
   - **奖励金额**: 设置积分奖励
   - **截止日期**: 可选，设置完成期限

3. 点击 **"发布"** 按钮
4. 系统自动扣除相应积分
5. 发布成功，跳转到列表页

### 发布技巧

✅ **好的标题**:
- "AI客服智能体开发"
- "订单查询数据库优化"

❌ **不好的标题**:
- "帮我做个东西"
- "需要一个程序"

✅ **好的描述**:
- 明确说明需求
- 列出具体要求
- 提供示例或参考

❌ **不好的描述**:
- "随便做做"
- "你懂的"

## 🎁 领取悬赏

### 领取条件

- 必须登录账户
- 悬赏状态为"开放中"
- 具备所需技能

### 领取步骤

1. 在列表页或详情页找到感兴趣的悬赏
2. 点击 **"领取任务"** 按钮
3. 确认领取
4. 状态变为"已领取"
5. 开始完成任务

### 注意事项

⚠️ 领取后请尽快完成  
⚠️ 不要同时领取过多任务  
⚠️ 如无法完成，请联系发布者取消

---

## 📤 提交成果

### 提交流程

1. 进入"我领取的"页面
2. 找到已完成的任务
3. 点击 **"提交成果"** 按钮
4. 填写提交信息：
   - 成果链接（GitHub、文档等）
   - 说明文字（可选）

5. 点击 **"提交"** 按钮
6. 等待发布者验证

### 提交建议

✅ 提供清晰的成果链接  
✅ 附上简要说明  
✅ 确保代码/文档可访问  
✅ 检查是否有遗漏

---

## ✅ 验证成果

### 验证流程（仅发布者）

1. 进入"我发布的"页面
2. 找到待验证的任务
3. 点击查看提交的成果
4. 评估完成质量

5. 选择验证结果：
   - **批准**: 发放奖励（80%给领取者，20%平台）
   - **拒绝**: 返回说明，任务回到"已领取"状态

### 验证标准

✅ 功能完整实现  
✅ 代码质量良好  
✅ 文档清晰完整  
✅ 符合需求描述

## 💡 使用技巧

### 提高搜索效率

1. **使用搜索建议**
   - 输入1-2字即可看到建议
   - 减少打字时间 70%

2. **利用搜索历史**
   - 常用搜索一键重用
   - 避免重复输入

3. **关注热门搜索**
   - 发现大家都在搜什么
   - 找到热门任务

4. **组合过滤**
   - 技能 + 状态 + 关键词
   - 精准定位目标

### 提高成功率

1. **完善个人资料**
   - 展示技能和经验
   - 增加信任度

2. **仔细阅读需求**
   - 确保理解要求
   - 避免误解

3. **及时沟通**
   - 有疑问主动联系
   - 保持良好沟通

4. **按时交付**
   - 遵守截止日期
   - 提前完成更佳

## ❓ 常见问题

### Q1: 如何获得更多积分？

A: 通过以下方式获得积分：
- 完成悬赏任务获得奖励
- 参与平台活动
- 邀请好友注册

### Q2: 悬赏被拒绝了怎么办？

A: 
1. 查看拒绝原因
2. 与发布者沟通修改
3. 重新提交成果
4. 或放弃任务（不影响信用）

### Q3: 可以取消已领取的任务吗？

A: 
- 可以，但需要发布者同意
- 联系发布者说明原因
- 发布者可以取消任务

### Q4: 搜索结果不准确怎么办？

A: 
1. 尝试不同的关键词
2. 使用更具体的词汇
3. 结合技能过滤
4. 调整状态过滤

### Q5: 为什么看不到搜索建议？

A: 可能原因：
- 输入少于1个字符
- 没有匹配的悬赏
- 网络连接问题
- 尝试刷新页面
