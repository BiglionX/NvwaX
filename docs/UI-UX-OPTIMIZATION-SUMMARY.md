## 💡 核心设计理念

### 1. 紫罗兰渐变品牌色
```tsx
// 主色调
from-violet-500 to-purple-600

// 辅助色
from-blue-500 to-indigo-600

// 状态色
成功: from-green-500 to-emerald-600
警告: from-orange-500 to-amber-600
危险: from-red-500 to-rose-600
```

### 2. 精致的圆角系统
```
按钮/输入框: rounded-lg (8px)
卡片/面板:   rounded-xl (12px)
模态框/弹窗: rounded-2xl (16px)
头像/图标:   rounded-full
```

### 3. 彩色阴影增强品牌感
```tsx
shadow-lg shadow-violet-200/50 dark:shadow-violet-900/30
hover:shadow-xl hover:shadow-violet-300/50
```

### 4. 流畅的交互动画
```tsx
// 页面过渡
initial={{ opacity: 0, y: 20 }}
animate={{ opacity: 1, y: 0 }}

// 按钮交互
whileHover={{ scale: 1.02 }}
whileTap={{ scale: 0.98 }}

// 卡片悬停
hover:-translate-y-1 hover:shadow-xl
```

### 5. 完善的无障碍支持
- 键盘导航友好
- 屏幕阅读器兼容
- WCAG AA 合规
- 焦点管理完善
