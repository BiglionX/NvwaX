**A**: 它们被移动到 `docs-archive/` 目录了，仍然可以访问。查看 [ARCHIVE-INDEX.md](./docs-archive/ARCHIVE-INDEX.md) 找到它们。

### Q: 我需要查看某个阶段性报告怎么办？

**A**: 
1. 打开 [ARCHIVE-INDEX.md](./docs-archive/ARCHIVE-INDEX.md)
2. 找到对应的分类
3. 直接访问该文档

或者在命令行中：
```bash
ls docs-archive/phase-reports/
```

### Q: 我可以恢复某个归档文档吗？

**A**: 当然可以！归档文档仍然在 Git 中跟踪，只是移动了位置。如果需要，可以移回原处。

### Q: 文档太多，我该看哪些？

**A**: 
- **新用户**: 只看根目录的 README 和 GETTING-STARTED

- **研究者**: 可以查看所有文档，包括归档的

- **快速打开文件**: `Ctrl+P` → 输入文件名
- **全局搜索**: `Ctrl+Shift+F` → 搜索内容
- **文件树搜索**: `Ctrl+Shift+E` → 展开资源管理器

### 命令行技巧

```bash

# 查找特定主题的文档
find . -name "*SDK*" -type f
```
