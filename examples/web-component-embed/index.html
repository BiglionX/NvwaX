🎨 NvwaX Web Component Examples

See how to embed NvwaX components in your applications

📦 Agent Marketplace

Embed the full Agent Marketplace with search and filtering capabilities.

<nvwax-agent-marketplace api-key="your-api-key" base-url="http://localhost:3001"> </nvwax-agent-marketplace> <script> const marketplace = document.querySelector('nvwax-agent-marketplace'); marketplace.addEventListener('skill-selected', (event) => { console.log('Selected agent:', event.detail.skill); }); </script>

🎨 Custom Theme

Customize the appearance using CSS custom properties.

<style> nvwax-agent-marketplace.custom-theme { --primary-color: #8b5cf6; --secondary-color: #ec4899; --card-shadow: 0 10px 15px -3px rgba(0, 0, 0, 0.1); } </style> <nvwax-agent-marketplace class="custom-theme" api-key="your-api-key"> </nvwax-agent-marketplace>

🛠️ Agent Studio

Embed the low-code Agent builder directly in your application.

<nvwax-agent-studio api-key="your-api-key" base-url="http://localhost:3001/studio" @agent-saved="handleSave" @agent-published="handlePublish"> </nvwax-agent-studio> <script> const studio = document.querySelector('nvwax-agent-studio'); studio.addEventListener('agent-saved', (event) => { console.log('Agent saved:', event.detail); }); studio.addEventListener('agent-published', (event) => { console.log('Agent published:', event.detail); }); </script>

⚙️ Programmatic Control

Control the components programmatically using their public API.

// Load a specific template studio.loadTemplate('template-id-123'); // Save current agent studio.saveAgent(); // Publish agent studio.publishAgent(); // Get current configuration const config = await studio.getAgentConfig();
